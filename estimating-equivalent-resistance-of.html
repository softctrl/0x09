Hi. in a course of a basic physics, we can learn to calculate a equivalent resistance of the association of resistors. But, we cant see how this can be proved in the real life. To this proof of concept we will use two resistors, one of 330Ω and other of 1000Ω or 1kΩ, both  with 5% of variation.
First, we will connect this two resistors in serie and we will verify the final value using a simple multimeter:
How you can see, when we put the resistor in serie, the equivalent resistance will be the sum of all those single values of each resistor: Req = R1 + R2 + ... + Rn Req = 330Ω + 1000Ω = 1330Ω Now, we will link those resistors in parallel mode,
In this case the equation is a little more complex, or not :) 1/Req = 1/R1 + 1/R2 + ... + 1/Rn 1/Req = 1/330Ω + 1/1000Ω = (330Ω + 1000Ω) / (330Ω . 1000Ω) Then: Req = (330Ω . 1000Ω) / (330Ω + 1000Ω) = 248.120300752Ω = 248Ω
Also, this particular case we can simply do this: Req = (R1Ω . R2Ω) / (R1Ω + R2Ω) A very usefull rule is, when we connect many resistors in serie, the value of the equivalent resistance allways will be less then the value of the resistor with less value. In your sample, the less valuable resistor is the resistor with 330Ω, and the equivalent resistance when connection is in parallel was the value 248Ω < 330Ω. I use this rule to validate my result in circuits more complex.
