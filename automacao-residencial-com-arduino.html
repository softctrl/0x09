Basicamente o que este circuito faz é que ao alimentarmos a base do nosso transistor o mesmo permitirá que a corrente flua fechando o circuito para que o nosso relê seja ativado. Este por sua vez irá fechar o contato e então um equipamento de tensão 115V com corrente máxima de 15A poderá ser acionado.

Apesar de eu não ter incluído o resistor entre a base do transistor e a fonte de 5V, posso dizer que não há problemas pois o circuito esta totalmente funcional.
Agora que o protótipo está montado podemos desenvolver um pequeno sketch para controlar nosso "módulo relê":
 int LED_13 = 13; // Led que usaremos para indicar que o relê foi acionado



void setup() {                
  pinMode(LED_13, OUTPUT);
  pinMode(A0, OUTPUT); // Tratar o pino analogico A0 como se fosse digital
}

void loop() {
  
  // Acionar o modulo relê ao passo que a luz do led ficara aceso indicando relê acionado:
  digitalWrite(A0, HIGH);     // Acionar circuito com o rele
  digitalWrite(LED_13, HIGH); // Ligar LED
  
  delay(2000); // Aguarde 2 segundos
  // Desativar o circuito do rele e desligar o led:  
  digitalWrite(A0, LOW);
  digitalWrite(LED_13, LOW);
  delay(2000); // Aguarde 2 segundos
  
}
